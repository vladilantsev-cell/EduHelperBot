# Функции по ТЗ

# Родитель
- 📚 Расписание (день / неделя / месяц)
- 💰 Финансы (задолженности + история)
- 🎉 Мероприятия (просмотр + регистрация)
- 📊 Успеваемость (оценки + средний балл)
- 📄 Справки (об обучении / налоговый вычет)
- 📰 Новости школы
- 👨‍🏫 Запрос обратной связи от учителя
- 📅 Запись на встречу

# Ученик
- ⭐ Movavi Boost (баланс + история + магазин)
- 🎉 Мероприятия (просмотр + регистрация)
- 📚 Расписание
- 📖 Учебные материалы (ссылки)
- 👨‍🏫 Написать учителю

# Администратор
- 📋 Список всех пользователей
- 📢 Рассылка новостей (всем подписчикам)
- 🗓 Добавить мероприятие
- ⭐ Начислить бусты ученику
- 🛒 Добавить товар в магазин
- 📝 Добавить занятие в расписание
- Получает уведомления о всех запросах от пользователей
