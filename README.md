# EduHelper - Telegram бот для ИТ школы Movavi

## Описание
Многофункциональный чат-бот для взаимодействия родителей и учеников со школой.

## Функциональность
### Для родителей
- Просмотр расписания
- Финансовый блок (задолженности, история платежей)
- Мероприятия (анонсы, регистрация)
- Успеваемость (оценки, отзывы)
- Заказ справок
- Новости школы
- Связь с учителем
- Запись на встречу

### Для учеников
- Система мотивации Movavi Boost
- Мероприятия
- Расписание уроков
- Учебные материалы
- Общение с преподавателями

## Установка и запуск
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать файл `.env` с токеном бота
4. Запустить: `python main.py`

## Технологии
- Python 3.8+
- aiogram 2.x
- SQLAlchemy
- SQLite
